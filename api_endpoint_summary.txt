System Status & Metadata
====================================================================================================

/api/v1/meta/build
----------------------------------------------------------------------------------------------------

  Method: GET
  Summary: Get build environment information
  Description: Returns the Go toolchain version, target OS, and architecture used to compile the server binary.

  Responses:
    200: OK
      Schema: BuildInfo (application/json)
    default: Error
      Schema: ErrorModel (application/problem+json)


/api/v1/meta/config
----------------------------------------------------------------------------------------------------

  Method: GET

  Responses:

      Schema: ConfigInfo (application/json)
