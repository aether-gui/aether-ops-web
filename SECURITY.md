# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Aether Ops Web, please report it privately using GitHub's vulnerability reporting form:

**[Report a vulnerability](https://github.com/aether-gui/aether-ops-web/security/advisories/new)**

Do not open a public issue for security vulnerabilities. Private reporting ensures the issue can be addressed before public disclosure.

We will acknowledge receipt within 72 hours and provide an initial assessment within one week.
